Manual 3DViewer v1.0

author{chesterh, cyndisig, cornichl}
date{Сентябрь 02 2023}


Общее описание

3DViewer v1.0 - программа для просмотра 3D моделей в каркасном виде, с возможностью вращения, масштабирования и перемещения моделей.


Предназначен для просмотра 3D моделей в каркасном виде.
Поддерживает вращение, масштабирование и перемещение моделей.
Поддерживает следующие операции:

Загрузка каркасных моделей из файла формата obj (поддержка только списка вершин и поверхностей);
Перемещение модели на заданное расстояние относительно осей X, Y, Z;
Поворот модели на заданное расстояние относительно осей X, Y, Z;
Масштабирование модели на заданное значение.
Программа должна позволять настраивать тип проекции (параллельная и центральная);
Программа позволяет настраивать тип (сплошная, пунктирная), цвет и толщину ребер, способ отображения (отсутствует, круг, квадрат), цвет и размер вершин;
Программа позволяет выбирать цвет фона;
Настройки должны сохраняться между перезапусками программы

Программа позволяет сохранять полученные ("отрендеренные") изображения в файл в форматах bmp и jpeg
Программа позволяет записывать небольшие "скринкасты" 


Установка

Скачайте репозиторий проекта;
Перейдите в терминале в папку src проекта;
Выполните \textbf{make install};
Откройте папку приложения на рабочем столе и запустите приложение.

Удаление

Перейдите в терминале в папку src проекта;
Выполните: \textbf{make uninstall};
Удалите папку проекта.
